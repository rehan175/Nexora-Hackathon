# **App Name**: NEXORA

## Core Features:

- Event Countdown & Info Display: Displays the real-time countdown to the hackathon, key event statistics, and general informational sections like timeline and rules.
- Track & Problem Statement Showcase: Presents detailed information for each hackathon track and their corresponding problem statements via interactive modals.
- User Registration Form: Allows participants to register for the hackathon by submitting personal and team details, including track selection and team size.
- Responsive Layout & Navigation: Ensures the application's interface adapts seamlessly to different screen sizes and provides a sticky navigation bar for easy access to sections.
- AI Problem Statement Assistant: A generative AI tool to assist hackathon organizers in creating novel and diverse problem statements tailored to specific tracks and themes, enhancing the challenge roster.
- Prizes and Awards Display: Dedicated sections highlighting the total prize pool and breakdown for each award category, designed to incentivize participation.

## Style Guidelines:

- The visual theme is a 'Digital Cyber Frontier' in a dark scheme. Primary color: a vibrant electric cyan (#00D4FF), for energy and emphasis, heavily used for interactive elements, highlights, and text glows. Background color: A very dark, desaturated blue-black (#080F11) to create depth and contrast for the bright elements. Accent color: A vibrant blue-green (#14AD7E), analogous to the primary, to provide secondary highlights and variation, also a distinct gold (#FFD700) for prize distinction.
- Headlines and prominent display text: 'Orbitron' (sans-serif, geometric) for its bold, futuristic, and impactful presence. Body text and subtitles: 'Rajdhani' (sans-serif, humanist) for legibility and a contemporary feel. Technical details and HUD elements: 'Share Tech Mono' (monospace) to convey a computerized, terminal-like aesthetic. Note: currently only Google Fonts are supported.
- Utilize a combination of illustrative emojis and Font Awesome icons, stylized with the application's signature cyan glow and shadows, to maintain a consistent futuristic, high-tech visual language across all interactive and informational elements.
- A structured cybernetic grid-like layout with distinct, full-width sections for information, responsive to screen size. Features modular components with chiseled borders and pseudo-elements to reinforce the tech aesthetic, ensuring content is visually framed.
- Implement subtle yet impactful futuristic transitions and micro-interactions. Key animations include text glows, rotating rings in the hero section, GSAP-driven scroll animations for elements fading in, and button ripple effects, all enhancing the dynamic, high-tech user experience.
